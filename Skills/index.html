Habilidades e Conhecimentos

HTML

  • Estrutura básica das páginas da web.
  • Define títulos, parágrafos, imagens, links e formulários.
  • Usa tags para organizar o conteúdo.
  • É o "esqueleto" de qualquer site.

CSS

  • Controla o estilo e a aparência da página.
  • Define cores, fontes, tamanhos e espaçamentos.
  • Permite criar layouts responsivos e modernos.
  • Separa o design da estrutura (HTML).

JavaScript

  • Linguagem de programação para interatividade.
  • Manipula o HTML e o CSS em tempo real (DOM).
  • Permite validações, animações e requisições a servidores.
  • Torna a página dinâmica e responsiva às ações do usuário.

    • React

    • Biblioteca JavaScript para criar interfaces de usuário.
    • Baseado em componentes reutilizáveis.
    • Usa o Virtual DOM para atualizar a tela de forma eficiente.
    • Facilita a criação de aplicações web dinâmicas e escaláveis.
